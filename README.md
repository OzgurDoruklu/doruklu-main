# doruklu.com

Bu proje, **doruklu.com** kök alan adını (root domain) barındıran GitHub Pages statik web sitesidir.

## Özellikler
- **Tasarım:** Modern, Dark Mode, Glassmorphism, CSS Animasyonları.
- **Odak:** Ailenin ana merkezi ve çocukların alt alan adlarına (subdomain) yönlendirme.
- **Hosting:** GitHub Pages (A Kayıtları ile doruklu.com üzerinden).
